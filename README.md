# arrow-debreu-equilibrium

Walrasian equilibrium solver for Arrow-Debreu exchange economies via tatonnement and Eisenberg-Gale convex programming.

## Quickstart

```bash
git clone https://github.com/abhi-wadhwa/arrow-debreu-equilibrium.git
cd arrow-debreu-equilibrium
make install
make run
make test
```

See `docs/` for the math.
